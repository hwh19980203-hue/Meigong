# 欢迎来到你的 LLM 驱动知识库

这个知识库基于 [[LLM]] 中描述的设计模式构建——一个由 AI 助手维护、持续积累的 Wiki 知识库。

## 快速开始

### 试试这些操作

**📥 摄入资料**
> "帮我处理这份资料"
> "读一下这篇文章，把它整合到知识库里"

**🔍 查询知识**
> "关于 X 我知道些什么？"
> "帮我总结一下关于 Y 的所有知识"

**🔧 维护检查**
> "检查一下知识库的健康状况"
> "看看有没有矛盾或过时的内容"

### 知识库结构

```
📁 raw/           ← 你存放原始资料的地方（文章、书籍笔记等）
📁 wiki/          ← AI 为你生成的 Wiki 知识库
   ├── index.md   ← 内容索引（自动维护）
   ├── log.md     ← 操作日志（自动记录）
   ├── entities/  ← 实体页面
   ├── concepts/  ← 概念页面
   └── sources/   ← 资料摘要
📄 CLAUDE.md      ← 给 AI 的工作指令（可共同进化）
```

### 小技巧

- 使用 **Obsidian Web Clipper** 快速保存网页到 `raw/articles/`
- 设置快捷键 **Ctrl+Shift+D** 下载文章中的图片到本地
- 用 **图谱视图** 查看知识之间的连接
- 好的 AI 回答可以保存到 Wiki 中，让知识持续积累
