# 原始资料库 (Raw Sources)

> 本目录存放**不可修改的原始资料**。LLM 可以读取但不会修改这些文件。
> 这是知识库的"源文档"——这是我们知识探索的起点。

## 目录结构

```
raw/
├── assets/        # 下载的图片/截图（本地化存储）
├── articles/      # 文章、博客、新闻
├── books/         # 书籍章节摘录
├── podcasts/      # 播客文稿/笔记
├── videos/        # 视频文稿/笔记
├── papers/        # 论文、研究报告
└── images/        # 截图/图表
```

## 使用方式

1. 使用 Obsidian Web Clipper 等工具将网页内容保存到 `raw/articles/`
2. 下载图片到 `raw/assets/`（在 Obsidian 设置中绑定快捷键 Ctrl+Shift+D）
3. 将书籍笔记、播客笔记等分类放入对应子目录
4. 告诉 LLM 处理新加入的原始资料 → LLM 会读取并整合到 Wiki 中

## 规则

- **不可修改**：原始资料一旦放入就不应手动编辑（保留原始内容）
- **命名规范**：使用有意义的中文/英文文件名，如 `如何构建知识库-2024.md`
- **来源标注**：尽量在文件开头标注来源链接和日期
