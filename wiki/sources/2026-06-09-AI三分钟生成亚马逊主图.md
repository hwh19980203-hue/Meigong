# 如何用AI三分钟生成一套亚马逊主图？

> **来源**：知无不言跨境电商社区  
> **原始文件**：[[raw/articles/2026-06-09-AI三分钟生成亚马逊主图.md]]  
> **核心工具**：ChatGPT (Image 2 / GPT-4o 图片生成能力)

---

## 内容概要

本文详细介绍了使用 ChatGPT 的 Image 2 功能快速生成亚马逊电商主图的完整方法论，覆盖从铺货卖家到精品卖家的不同需求层次，共 9 个模块。

---

## 核心内容

### 一、一句话出图（铺货卖家）

对于铺货/精铺卖家，只需要一条包含以下要素的提示词即可快速出图：
- 产品名字 + 用途 + 材质（含各部位）+ 结构 + 颜色 + 功能 + 尺寸

**工作流**：
1. 拍摄产品多角度照片
2. 让 AI 先详细描述产品材质与特点
3. 手动补充修正描述
4. 组合完整提示词 + 产品图发给 ChatGPT（需 Plus 会员 + Think 模式）
5. 一键生成 6 张图（白底主图 + 场景图 + 细节图 + 卖点图）

> 示例图片：![[raw/assets/ai-main-img-01.jpg]] ![[raw/assets/ai-main-img-02.jpg]]

### 二、产品精修技巧

**方案一：提示词精修**
- 通用模板：产品精修 + 纯净白底 + 正视图 + 3D 渲染 + 材质精准还原 + 清除瑕疵
- 适用于塑料、金属等材质

**方案二：线稿精修**
- 针对布艺产品的立体感问题，通过线稿约束形状
- 避免产品变形

> 示例图片：![[raw/assets/ai-main-img-10.jpg]] ![[raw/assets/ai-main-img-11.jpg]]

### 三、精细化出图（精品卖家）

一张一张出图，每张图精确控制：
1. 字体、风格参考
2. 排版布局参考
3. 产品参考（保持一致性）
4. 场景风格、模特要求、机位角度
5. 主打卖点标注
6. 小图标
7. 文字描述（大标题、副标题、小字）
8. 尺寸要求

> 示例图片：![[raw/assets/ai-main-img-12.jpg]] ![[raw/assets/ai-main-img-17.jpg]]

### 四、复刻竞品风格（不侵权）

将竞品图发给 AI，反推版式 Prompt（注意：反推版式布局而非产品细节）：
- 构图与比例
- 产品位置、占比、角度
- 模特位置与互动关系
- 文字/图标排版层级
- 细节图分布
- 背景与场景分区
- 视觉重心与留白
- 光线与色调

### 五、导出 PSD 格式

提示词要求 AI 分图层：商品主体 + 背景 + 文字 + 品牌 logo + 模特。  
**需要**：安装 PS 插件 + 对话框选 PS 和 Think 模式。

### 六、AI 图片检查要点

1. 产品一致性（细节、文字、螺丝等）
2. 产品比例是否合理（提示词中写清楚尺寸）
3. 文字/logo 拼写是否正确
4. 是否有 AI 自行拓展的内容

### 七、场景一致性

使用 Image 2 的全景图功能，可模拟 360° 真实场景。对家具卖家尤其友好，也可用于微调视角。

### 八、AI 图片真实感增强

**人物真实感**：
- 动作描写具体化（年龄、服饰、姿势、眼神方向）
- 增加真实皮肤纹理、轻微毛孔、自然肤色
- 避免过度完美的"AI 感"

**产品真实感**：
- 强调不能"飘"、底部接触表面、有自然阴影
- 产品表面有微细节

**场景真实感**：
- 层次感 + 生活痕迹（床单褶皱、枕头塌陷等）

**核心原则**：AI 的假来自于完美，真实来自于适度瑕疵。

### 九、AI 生图的局限性

- 颜色把控不足（即使 RGB 控制也难以完全一致）→ 需 PS 后期调色
- 产品细节不稳定（AI 可能增加/减少螺丝等）→ 需后期微调
- 非会员额度有限，一次只能生成一张，无法导出 PSD

## 关键提示词模板

| 场景 | 要点 |
|------|------|
| **一句话出图** | 产品名称+用途+材质+结构+颜色+功能+尺寸+文字强调 |
| **产品精修** | 产品精修,白底,正视图,3D渲染,材质还原,清除瑕疵 |
| **精细化出图** | 字体参考+排版参考+产品参考+场景风格+卖点+文字+尺寸 |
| **复刻竞品** | 反推版式Prompt：构图/位置/模特/文字层级/光影/色调 |
| **PSD导出** | 分图层分离：商品主体/背景/文字/logo/模特，PSD格式 |

- **ChatGPT**（需 Plus 会员，Think 模式）
- **PS 插件**（用于导出 PSD 和后期微调）
